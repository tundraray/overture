# Context Mode: Review

**Active during**: Phase 6 (Report Compilation), any document review or quality check

## Behavioral Rules

1. **Read ALL source documents thoroughly** — do not skim. Every doc may contain contradictions or insights
2. **Prioritize by severity** — Critical > High > Medium > Low. Report critical findings first.
3. **Detect contradictions** — if two documents disagree, flag it explicitly with both positions and evidence
4. **Verify source tiers** — check that Tier 3 data is not presented as Tier 1 anywhere
5. **Verify cross-references** — check that all referenced docs/sections actually exist
6. **Verify completeness** — check that all template sections are filled (no empty placeholders)
7. **Synthesize, don't summarize** — create new insights from combining multiple documents, don't just concatenate
8. **Confidence-filtered reporting** — only report findings with >80% confidence. Tag uncertain items as "[Low confidence — needs validation]"

## Quality Checklist (apply to every document)

- [ ] Pyramid Principle applied (conclusion first)
- [ ] All categorizations are MECE
- [ ] Action titles on every section
- [ ] Every data point has source tier tag
- [ ] No vague language (no "some", "many", "significant")
- [ ] Numbers are specific with units
- [ ] Cross-references to source documents present
- [ ] Assumptions explicitly stated

## Anti-Patterns

- Summarizing instead of synthesizing
- Ignoring contradictions between documents
- Accepting Tier 3 data without disclosure
- Reporting low-confidence findings without flagging them
- Missing documents in the source index
