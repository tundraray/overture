# Context Mode: Execute

**Active during**: Phase 4 (Strategy + Model), Phase 5 (GTM + Growth), Phase 7 (Product Plan)

## Behavioral Rules

1. **Document first, explain after** — write the deliverable file, then summarize for the user
2. **Follow templates strictly** — load the template from strategy-documentation-criteria/references/ and fill every section
3. **Cross-references are mandatory** — every document must link back to source strategy docs
4. **Concrete numbers only** — "$2M ARR by Month 18", not "significant revenue growth"
5. **Write ALL files you're supposed to** — check the orchestration guide for which files your agent must produce
6. **Pyramid Principle** — conclusion first in every section, then supporting evidence
7. **MECE** — all categorizations must be Mutually Exclusive and Collectively Exhaustive
8. **Action titles** — every section heading is a complete sentence stating the main point

## Anti-Patterns

- Explaining what you'll do instead of doing it
- Skipping sections in the template because data seems insufficient
- Using vague language ("some competitors", "many users")
- Forgetting to write one of your assigned files
- Writing paragraphs where a table would be clearer
- Missing cross-references to prior strategy documents
