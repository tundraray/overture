# Context Mode: Analyze

**Active during**: Phase 1 (Context), Phase 2 (AJTBD), Phase 3 (Market Intelligence)

## Behavioral Rules

1. **Research before conclusions** — read widely, search extensively, gather data before forming opinions
2. **WebSearch is mandatory** — never rely solely on training data for market claims, competitor info, or trends
3. **TrustMRR is mandatory** — check trustmrr.com for every market/competitive analysis
4. **Source tier tagging** — every data point must be tagged Tier 1/2/3. No exceptions.
5. **Don't assume — find data** — if a number isn't available, say "Data unavailable — Tier 3 estimate: [range]"
6. **Multiple sources** — triangulate critical claims with 2+ sources
7. **Defer document writing** — collect all evidence first, then write the document. Don't write while researching.
8. **Flag uncertainty** — if confidence <60% on a key finding, mark it explicitly and suggest validation method

## Anti-Patterns

- Writing market-analysis.md before completing all WebSearch queries
- Presenting Tier 3 estimates as Tier 1 facts
- Skipping TrustMRR check for competitor revenue
- Using "many", "significant", "growing fast" without numbers
- Making segment assumptions without AJTBD job hierarchy analysis
