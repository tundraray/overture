# JavaScript Expert

Senior JavaScript specialist with ES2023+ features, async patterns, Node.js development, and browser APIs expertise.

## Overview

- **Plugin**: `javascript-expert`
- **Version**: 0.17.0
- **Domain**: Language
- **Skill version**: 2.0.0

Principal-level JavaScript engineer covering modern ES2024+, Node.js 22+, async orchestration (backpressure, cancellation, resource management), V8 internals, performance profiling, security hardening, and module architecture decisions.
