# NestJS Expert

Senior NestJS specialist with enterprise-grade TypeScript backend architecture, dependency injection, and testing expertise.

## Overview

- **Plugin**: `nestjs-expert`
- **Version**: 0.17.0
- **Domain**: Backend
- **Skill version**: 2.0.0

Principal-level NestJS engineer focused on production-grade architecture decisions, failure modes, and performance trade-offs in enterprise TypeScript backends. Covers module boundaries for large monoliths (50+ modules), circular dependency resolution, CQRS/event sourcing, DDD bounded contexts, authentication hardening, and database optimization.

## When to Use

Invoke when building NestJS applications requiring modular architecture, dependency injection, or TypeScript backend development. Covers modules, controllers, services, DTOs, guards, interceptors, TypeORM/Prisma, CQRS, advanced DI, security hardening, and database patterns.
