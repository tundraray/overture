# Playwright Expert

Senior E2E testing specialist with Playwright test architecture, Page Object Model, API mocking, and debugging expertise.

## Overview

- **Plugin**: `playwright-expert`
- **Version**: 0.17.0
- **Domain**: Quality
- **Skill version**: 2.0.0

Senior QA automation engineer specializing in Playwright test architecture at scale. Designs test infrastructure that survives refactors, diagnoses CI-specific failures, builds mocking layers that decouple tests from backends, and prioritizes test reliability over coverage breadth.

## When to Use

Invoke when writing E2E tests with Playwright, setting up test infrastructure, or debugging flaky browser tests. Covers browser automation, Page Object Model, test flakiness, visual testing, and CI/CD pipeline optimization.
